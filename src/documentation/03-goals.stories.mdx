# Goals

## The core goals of this project are to:

1. 1) Speed up design and development velocity
2. 2) Help create consistent, beautiful, and usable UI in our applications
3. 3) Promote best practices for accessibility, internationalization, and responsive web design

## We hope to accomplish these goals by:

1. 1) Reducing the number of decisions needed when iterating on UI
2. 2) Reducing the amount of code duplication in our apps
3. 3) Serving as the standard for trigo&apos;s visual language
4. 4) Providing easy-to-use and extensible components for anyone to consume
