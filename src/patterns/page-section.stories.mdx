# PageSection

A `PageSection` is semantically used as a seperator on a single page. A single page can contain 
multiple PageSections and provides a semantic separation of different component groups. 
Its used to wrap other components collections, so its simply a container with some
shadow arround and a header title.

## Example with ViewSection as Children

        <PageSection

                                {
                                    field: 'customerNumber',

                                        min: 100000,
